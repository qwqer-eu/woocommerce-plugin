== Description ==
QWQER shipping for Woocommerce
This plugin gives you the ability to use QWQER shipping in your store.

== Installation ==
After activating the plugin you will need to add the shipping method (QWQER Shipping Method) in the store settings and enter your data (Api token, Trading-Point ID, category, phone number).
